# ImitatorGame - 模仿者游戏

Paper 1.21.5 插件，复刻**第五人格（Identity V）「模仿者游戏」**模式。

这是一个类似 Among Us 的社交推理游戏，10 名玩家分为三大阵营在地图内完成各自目标。

## 游戏规则

### 阵营构成（标准模式 10 人）

| 阵营 | 人数 | 胜利条件 |
|------|------|----------|
| 侦探团 | 6 | 完成所有任务 OR 投票淘汰所有模仿者 |
| 模仿者 | 2 | 模仿者人数 ≥ 幸存者一半 |
| 神秘客 | 2 | 各自完成独特个人目标 |

### 游戏流程

```
大厅等待 → 开始倒计时 → 身份揭示 → 自由行动（做任务/击杀）
    → 发现尸体/使用电台 → 会议讨论 → 投票 → 淘汰或继续
    → 重复直到某方胜利
```

### 角色列表

**侦探团（6选6）：**
| 角色 | 物品 | 能力 |
|------|------|------|
| 侦探 | 指南针 | 右键玩家查看阵营颜色 |
| 治安官 | 金剑 | 击杀任何人，杀错则自己死 |
| 猎人 | 钻石剑 | 击杀任何人，仅1次 |
| 哨兵 | 铃铛 | 死亡时自动召开紧急会议 |
| 锁匠 | 线 | 开关秘密通道铁门 |
| 香料师 | 烈焰粉 | 标记玩家获得追踪发光效果 |

**模仿者（4选2）：**
| 角色 | 物品 | 能力 |
|------|------|------|
| 神偷 | 药水 | 10秒隐身+速度 |
| 千面人 | 玩家头颅 | 伪装成其他玩家外观 |
| 阴谋家 | 书与笔 | 会议中猜身份，猜对目标死/猜错自己死 |
| 烟火师 | TNT | 放置定时炸弹 |

**神秘客（3选2）：**
| 角色 | 物品 | 能力 |
|------|------|------|
| 愚人 | 无 | 被投票淘汰时个人胜利 |
| 流浪汉 | 纸 | 与N个不同玩家互动获胜 |
| 送货员 | 命名牌 | 向所有玩家投递包裹获胜 |

### 特殊事件

- **停电**：非模仿者致盲60秒，前往配电室修复
- **水淹**：75秒倒计时，前往两个浴室关闭阀门，到期模仿者直接获胜

## 命令

| 命令 | 说明 |
|------|------|
| `/ig join` | 加入游戏大厅 |
| `/ig leave` | 离开大厅 |
| `/ig start` | 开始游戏（管理员） |
| `/ig stop` | 强制结束（管理员） |
| `/ig list` | 查看玩家列表 |
| `/ig reload` | 重载配置 |
| `/ig setspawn <lobby\|meeting\|game>` | 设置生成点 |
| `/ig addtask <type> <room>` | 添加任务位置 |
| `/ig adddoor` | 添加秘密通道门 |
| `/vote <player\|skip>` | 会议中投票 |
| `/report` | 报告尸体 |
| `/radio` | 使用电台呼叫会议 |

## 构建

```bash
# 需要 JDK 21+ 和 Maven 3.9+
mvn clean package
```

生成的 JAR 文件在 `target/ImitatorGame-1.0.0.jar`，放入服务器 `plugins/` 目录即可。

## 配置

插件首次加载时会在 `plugins/ImitatorGame/config.yml` 生成默认配置，包含所有游戏参数和地图位置设置。

## 项目结构

```
src/main/java/com/imitatorgame/
├── ImitatorGamePlugin.java        # 主插件类
├── config/                         # 配置管理
├── game/                           # 游戏状态机、会话管理、胜利条件
├── role/                           # 角色、阵营、能力接口
├── meeting/                        # 会议管理、投票系统
├── task/                           # 任务系统
├── event/                          # 特殊事件（停电、水淹）
├── listener/                       # 事件监听器
├── command/                        # 命令处理
├── ui/                             # 计分板、BossBar、消息
└── util/                           # 工具类
```

## 需求

- Paper 1.21.5 (26.1.2) 或更高版本
- Java 21+
