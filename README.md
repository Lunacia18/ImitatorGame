# ImitatorGame — 模仿者游戏

Paper 1.21.5 插件，复刻**第五人格「模仿者游戏」**模式。10 人社交推理，三大阵营各怀目标。

## 快速开始

1. 将 `ImitatorGame-1.0.0.jar` 放入服务器 `plugins/` 目录
2. 启动服务器，等待插件加载
3. 玩家进入服务器自动传送到虚空大厅
4. 每位玩家输入 `/ready` 准备
5. 准备人数达标后 **5 秒倒计时**开始，屏幕中央显示数字
6. 倒计时结束自动传送至 50×50 游戏地图

## 游戏规则

### 阵营与胜利条件

| 阵营 | 人数 | 胜利条件 |
|------|------|----------|
| **侦探团** | 6 | 完成 55 个任务 **或** 投票淘汰所有模仿者 |
| **模仿者** | 2 | 使模仿者人数 ≥ 幸存者一半 |
| **神秘客** | 2 | 达成个人目标（独立胜利） |

> 若场上存在**送货员**，侦探团和模仿者的普通胜利条件被阻断。需淘汰送货员或进入急速时刻。

### 游戏流程

```
大厅准备 → 5秒倒计时(Title居中) → 随机出生在50×50平台
    → 自由行动(做任务/击杀/隐蔽)
    → 右键制图台或骨块报告 → 传送到会议室环状开会
    → GUI投票 → 淘汰或继续 → 循环
    → 一方达成胜利条件 → 全员回到虚空大厅
```

### 地图

- **50×50 石砖平台**，萤石每 5 格嵌入地面照明，永昼
- 平台南侧独立 **会议室**（10×10×5，黑石+石英）
- **方块保护**：游戏中不可挖掘/放置（管理员除外）
- **制图台**（会议室附近）：右键呼叫会议
- **6 个随机磁石**：侦探团任务点，右键完成，5 秒冷却
- **两个铁门**（平台对角）：模仿者/锁匠可进入隐蔽
- **熔炉 + 工作台**：随机分布在中心 10×10 范围

## 角色

### 侦探团（6人）

| 角色 | 物品 | 左键 | 右键 | 冷却 |
|------|------|------|------|------|
| **侦探** | 指南针 | — | 调查玩家阵营 | 24s，全局 2 次 |
| **治安官** | 金剑 | **一刀毙命**，杀错自己死 | — | 24s |
| **猎人** | 钻石剑 | **一刀毙命** | — | 24s |
| **哨兵** | 无（被动） | — | — | — |
| **锁匠** | 线 | — | 开关铁门 / 潜行锁定15s | — |
| **香料师** | 烈焰粉 | — | 标记玩家发光 60s | 24s |

- **哨兵**：被击杀时击杀者强制拉起尸体开会
- **锁匠**：进入铁门隐蔽点 + 锁定铁门 15 秒

### 模仿者（2人）

| 角色 | 物品 | 左键 | 右键 | 冷却 |
|------|------|------|------|------|
| **神偷** | 药水 | 铁剑一刀 | 隐身+加速 10s | 24s |
| **千面人** | 头颅 | 铁剑一刀 | 记录→变装 30s | 24s |
| **阴谋家** | 书与笔 | 铁剑一刀 | 会议中猜身份 | 24s |
| **烟火师** | TNT | — | 造炸弹(20s)→右键传给别人 | 24s |

- 所有模仿者（除烟火师）配备**铁剑，一刀毙命**
- **千面人**：右键记录目标→再右键变装（同皮肤+同名称），持续 30 秒，期间可杀人
- **烟火师**：造炸弹后 20 秒倒计时，<11 秒他人可见烟花，0 秒爆炸 → 无骨块，脚下留煤炭块（会议后恢复）。**没有刀**。
- **模仿者额外物品**：信标（停电装置）、水桶（水阀装置），右键触发事件

### 神秘客（2人）

| 角色 | 物品 | 能力 | 胜利条件 |
|------|------|------|----------|
| **愚人** | 无 | — | 被投票淘汰 |
| **流浪汉** | 纸 | 右键互动不同玩家 | 与 ≥5 个不同玩家互动 |
| **送货员** | 命名牌 | 左键吞人入腹 | 急速时刻存活或成为最后自由者 |

- **送货员**：无限吞人，吞入者立刻隐身+视角绑定送货员。被吞者在会议召开时死亡，送货员被击杀则吞入者掉出。CD：32s→28s→24s→20s（此后永久20s，不被会议重置）。存在时阻断侦探团/模仿者普通胜利。
- **急速时刻**：≤3 人自由+送货员存活 → 65 秒倒计时，会议锁定，淘汰送货员否则它获胜

## 会议系统

- **触发方式**：右键制图台 / 右键骨块尸体 / 哨兵被动
- **传送到会议室**，环状站位，30 秒讨论 + 20 秒投票
- **GUI 投票**：背包界面点击玩家头颅投票，屏障=弃权，超时=自动弃权
- **淘汰条件**：最高票 > 弃权数 + 无平票
- **结果全屏 Title** 展示给所有人（含旁观者）
- 会议结束后**自动回到原位置**

## 特殊事件

| 事件 | 触发 | 效果 | 修复方式 | 时长 |
|------|------|------|----------|------|
| **停电** | 模仿者右键信标 | 非模仿者致盲+黑暗 | 右键工作台 25 次 | 无限（修为止） |
| **水阀** | 模仿者右键水桶 | 75 秒倒计时 BossBar | 右键熔炉 25 次 | 75 秒 |

- **互斥**：两个事件不能同时触发
- 修复时工作台/熔炉切换为**红石灯发光**，点击显示"还需 X 下"

## 铁门隐蔽点

- 平台两角**铁门**，模仿者/锁匠右键进入
- 进入后：**隐身 + 无法移动 + 无碰撞箱**
- **Shift** 退出并恢复自由行动
- 锁匠**潜行 + 右键铁门** = 锁定 15 秒（禁止进出）

## 其他机制

- **隐身手持**：所有玩家在他人视角中始终空手
- **死亡**：不掉落物品，生成骨块尸体+名牌，进入旁观模式，信标右键回大厅
- **旁观者**：会议中可自由飞行，可提前回大厅
- **游戏结束**：全员自动回虚空大厅，新玩家需等下一局

## 命令

| 命令 | 说明 |
|------|------|
| `/ready` | 准备（人数达标自动开局） |
| `/ig start` | 管理员强制开局 |
| `/ig stop` | 管理员强制结束 |
| `/ig list` | 查看玩家列表 |
| `/ig setspawn <lobby\|meeting\|game>` | 设置生成点 |
| `/ig addtask` | 添加任务点 |
| `/ig reload` | 重载配置 |

## 配置

`config.yml` 中可调整所有游戏参数：人数、冷却时间、事件时长等。

## 构建

```bash
# JDK 21+ / Paper API 1.21.5
mvn clean package
```
